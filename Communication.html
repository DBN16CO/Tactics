Communication.routehelper module¶

This file will handle all routed methods managing static data All methods must follow the following standards:

Inputs - they will all take in only one input: the JSON data

Outputs - they will pass back a formatted JSON response object which will detail the success or failure of the command as well as any other necessary information regarding the command.

Communication.routehelper.formJsonResult(error_msg, data=None)¶

Creates the basic response for any method that must return a JSON response.

Parameters:
  • error_msg (String) – The error message to be printed if the function did not execute sucessfuly.
  • data (Dictionary) – The inputted JSON command data that resulted in an error.
Return type:

Dictionary
Returns:

The response which can be sent back to the front-end user.

Note: If the result is a success, additional values may need to be added to the response dictionary.
